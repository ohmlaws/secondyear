<h3 id="q61">Q61: Calculate the average pitch ($Y_A$) for retrogressive wave winding, if No. of armature conductor =14, No. of slots=7, No. of poles 2</h3>
<ul>
<li onclick="checkAnswer(this, false)" data-correct="false">4</li>
<li onclick="checkAnswer(this, true)" data-correct="true">6</li>
<li onclick="checkAnswer(this, false)" data-correct="false">8</li>
<li onclick="checkAnswer(this, false)" data-correct="false">14</li>
</ul>
<details>
  <summary>Show Calculation</summary>

$$Y_A = \frac{Z \pm 2}{P}$$

Where:
* $Z$ = Number of armature conductors
* $P$ = Number of poles
* For **Progressive** winding, use $+2$
* For **Retrogressive** winding, use $-2$

**Given:**
* Number of conductors ($Z$) = $14$
* Number of poles ($P$) = $2$
* Winding type = Retrogressive

**Calculation:**

Since it is a retrogressive winding, we use the minus sign:

$$Y_A = \frac{Z - 2}{P}$$

$$Y_A = \frac{14 - 2}{2}$$

$$Y_A = \frac{12}{2}$$

$$Y_A = 6$$

</details>
